# SipBattle
Sip Battle is a turn-based 1v1 strategy game on an 8x10 board. Players battle the computer using 5 drinks: BlueBull, MoonBucks Coffee, Latorade, Phd.Pepper, and Kelvin. Each moves up to 2 spaces in any direction and has unique attacks. Eliminated drinks are replaced until one side loses all 5 and the winner is crowned.

# Sip Battle

Sip Battle is a Java turn-based strategy game where beverage cans battle using unique movement and attack styles.

## Built With
- Java
- Swing
- AWT / Java2D

## Features
- Custom vending-machine menu UI
- Fullscreen responsive gameplay
- AI opponent with difficulty levels
- Animated winner banner
- Unique can characters

## Run

javac *.java
java Main
